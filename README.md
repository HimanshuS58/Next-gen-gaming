# Next-gen-gaming
This project is an interactive gaming website built using React.js, Tailwind CSS, GSAP, and Vite, inspired by the award-winning design of Zentry.com. It focuses on creating a smooth and engaging user experience through scroll-based animations, fullscreen video transitions, custom clip-path effects, and responsive layouts. The goal was to recreate the premium feel of modern gaming websites while learning advanced frontend animation techniques and component-driven development.

The application features an animated floating navbar, interactive hero videos, reusable animated text components, GSAP ScrollTrigger animations, responsive Bento grid layouts with custom 3D tilt effects, mouse-driven image interactions, and audio controls for added interactivity. The project is built with reusable React components, custom Tailwind utilities, and a mobile-first approach to ensure clean code, easy maintenance, and a consistent experience across different screen sizes.
